Just Some Item, Mate

The BB-Edit Help Overlay!™

Press CTRL + L, choose a savefile, and press PARSE

If everything went well, the buttons with brothers' names and titles should appear below the path input. Press those to load your mercs and edit all kinds of stuff. But, if no mercs loaded, it means that they (or just one of them) have items that are not present in hex dictionary.

To edit perks - click on relevant perk icon.
To edit traits click on trait icon to remove it, or on +- icon to open trait list.
To change brother's background click on first icon in the trait field.
To edit an item click on it's icon in inventory field.

    Hotkeys:
  • F1 - - open this overlay.
  • ESC - - close topmost overlay (including this one).
  • CTRL + L - - open file selection dialog.
  • CTRL + S - - save current brother.
